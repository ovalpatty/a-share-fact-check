# Playbook 06 · 话术/诱导信号标记

**断言类型**：诱导话术、情绪煽动、暗示特权信息。**不走五步契约**——本类型不核真伪、无需外部证据，是纯模式识别，直接从 Step 2 的④类标签进入本手册。

## 识别模式（种子库，摸底后扩充）
- **紧迫感制造**："最后上车""抓紧""错过再等一年"
- **特权信息暗示**："内部消息""小道消息""机构才知道"
  - **技术崇拜变体**（同类新马甲，不另开类）："AI 算法信号""我的算力正在共振""只等模型信号"——内核与"内部消息"完全一致：暗示作者掌握普通散户不具备的信息/技术优势，制造信息差幻觉。归入本类当变体，**不新开一类**。
- **从众操纵**："游资抢筹""机构抢筹""聪明钱进场"
- **确定性话术**："稳赚""必涨""闭眼买""翻倍在即"
- **模糊权威**："据业内人士""有分析认为"（无具名来源）
  - **专业共同体暗示变体**（同类新马甲，不另开类）："我们团队连夜复盘""与多家机构交换了意见""连夜整理出一份报告"——无团队名、无机构名、无报告原文，用"专业共同体"的姿态借权威，内核仍是模糊权威。归入本类当变体。
  - **人格权威变体**（同类新马甲）："我是柚子出身""我经历过那个时代，我有话语权""船长的实力有目共睹"——以**人格/身份/资历**替代机构资质与可核分析框架作信任锚点，让读者信的是"这人是高手"而非任何可验证的东西。归入模糊权威当变体。
- **稀缺制造**："独家战法命名"（"主升断金战法""XX体系"）暗示外人得不到的秘技；"第N次亮剑""连胜"制造独家方法论幻觉。

**收敛纪律**：新话术优先认成已有类的新马甲，而非新开类。每出现一个新词就加一类，模式库会爆炸；认成变体，库才收敛。

## 结构级模式 · 匿名标的 / 钓鱼式荐股（whole-document，非单句）
这是比单句话术更高维的操纵**结构**，独立识别：
- **特征**：全文推荐的"核心标的"**从不具名**，只给"某公司""核心中的核心""关键供应商"，配一堆**具体到诱人但无法对账**的线索（"业绩增200%""获35家机构调研""华为间接持仓1300万股"）——线索精确到让读者相信"作者知道是谁"，但精确不到能独立验证。
- **商业模式**：免费内容用真事实建信任 → 匿名标的制造好奇 → 导流付费报告/私域拿标的名。
- **处理**：所有匿名线索判 ⚠️（主体不明）；触发 SKILL.md《匿名标的封顶规则》（全局可信度封顶 + 判词强制点破引流结构）。**凡不具名的"核心标的"推荐，不论附多少精确数据，结论一律"不可核、不可跟"。**

## 处理
逐条标 ⚠️风险信号，归入报告"话术与风险标记"段。涉"内部消息"等附一句信披合规**提示**（仅提示，不判违规——见 playbook 05 同款边界）。
